⇦Django CheatSheet

First of all, Run the following command to install django-admin-list-filter-dropdown module

pip install django-admin-list-filter-dropdown

Now go to the setting.py file of your root app. And place the following in your INSTALLED_APPS enclosing with single quotation mark.

django_admin_listfilter_dropdown

Go to the admin.py file of your app, import the module and apply the list_filter list like following coding snippet.

from django_admin_listfilter_dropdown.filters import ( DropdownFilter, ChoiceDropdownFilter, RelatedDropdownFilter )

class modelClass(admin.ModelAdmin):
    list_filter = [
                   ("gender_type", DropdownFilter),
                   ("area_type", DropdownFilter),
                   ("thana", DropdownFilter)
                   ]
    class Meta:
        model = SpatialData

admin.site.register(SpatialData, modelClass)

Save the code and re-run the project. Now you will see filter like the image below.
